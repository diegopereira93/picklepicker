# Core Web Vitals Optimization — Pitfalls Research

## Common Mistakes
- Over-optimization without measurement
- Breaking functionality for marginal gains
- Neglecting mobile performance
- Third-party script bloat

## Integration Pitfalls
- Conflicts between optimization techniques
- Breaking existing features during refactoring
- Lighthouse score ≠ real user experience

## Prevention Strategies
- Measure before optimizing
- A/B test changes
- Monitor Core Web Vitals in production
- Gradual rollout
