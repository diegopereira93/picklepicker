    1. Create `frontend/src/app/api/track/route.ts` (Edge runtime):
       ```typescript
       export const runtime = 'edge';
       ```
       - POST handler accepting JSON body:
         ```typescript
         interface TrackPayload {
           paddle_id: number;
           retailer: string;
           user_id?: string;  // anonymous UUID from localStorage
           timestamp: string; // ISO 8601
           utm_source?: string;
           utm_medium?: string;
           utm_campaign?: string;
           utm_content?: string;
           page_url: string;
         }
         ```
       - Validate: paddle_id and retailer are required, else return 400
       - Log structured JSON to stdout (Edge logs in Vercel):
         ```typescript
         console.log(JSON.stringify({ event: 'affiliate_click', ...payload }));
         ```
       - Return 204 No Content (fire-and-forget from client perspective)
       - Future: forward to analytics service or database

    2. Create `frontend/src/lib/tracking.ts`:
       - `trackAffiliateClick(params: { paddle_id: number, retailer: string, affiliate_url: string }): void`
         - Gets user_id from `getOrCreateUserId()` (from profile.ts)
         - Extracts UTM params from `window.location.search`
         - Fires `fetch('/api/track', { method: 'POST', body, keepalive: true })` -- fire-and-forget, no await
         - Catches errors silently (console.warn only -- never block user navigation)
       - `appendUtmParams(affiliateUrl: string): string`
         - Reads UTM params (utm_source, utm_medium, utm_campaign, utm_content, utm_term) from `window.location.search`
         - Appends to affiliate URL using URL API, skipping any already present on affiliate URL
         - Returns modified URL string
       - `extractRetailer(url: string): string`
         - Extracts retailer name from affiliate URL domain (mercadolivre.com.br -> "mercadolivre", etc.)

    3. Create `frontend/src/components/shared/affiliate-link.tsx`:
       - `AffiliateLink` component:
         - Props: `{ href: string, paddle_id: number, retailer?: string, children: ReactNode, className?: string }`
         - Renders `<a href={appendUtmParams(href)} target="_blank" rel="noopener noreferrer">`
         - onClick: calls `trackAffiliateClick({ paddle_id, retailer, affiliate_url: href })` before default navigation
         - Styled as shadcn Button variant="default" or custom styling via className
       - `FtcDisclosure` component:
         - Renders: "PickleIQ pode receber comissoes por compras realizadas atraves de links de afiliados nesta pagina."
         - Styled as small muted text with info icon
         - Must appear above the first affiliate link on any page that has them
