<objective>
Build the admin panel with /admin/queue (review queue for dedup/spec/price anomalies) and /admin/catalog (paddle catalog CRUD), protected by ADMIN_SECRET. Route Handlers proxy requests to FastAPI backend with authorization headers.

Purpose: The admin panel enables the solo founder to manage the data quality pipeline -- resolving duplicate paddles, unmatched specs, and price anomalies flagged by the automated crawlers. Without it, bad data propagates to users.
Output: /admin/queue and /admin/catalog pages with auth guard, API proxy routes, and action UI.
</objective>
