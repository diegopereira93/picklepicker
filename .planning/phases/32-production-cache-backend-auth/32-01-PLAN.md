  <action>Replace the ENTIRE contents of `backend/app/cache.py` with this implementation. The new class must:

1. **Import block**: `import json`, `import time`, `import hashlib`, `from typing import Optional, Dict, Any`, `import redis.asyncio as redis`

2. **`__init__`**: Accept `redis_url: Optional[str] = None`. Initialize `self.redis_url`, `self._redis: Optional[redis.Redis] = None`, `self._memory_cache: Dict[str, Any] = {}`, `self._memory_ttl: Dict[str, float] = {}`.

3. **`_ensure_client()`**: Private async method. If `self._redis is None and self.redis_url is not None`, initialize `self._redis = await redis.from_url(self.redis_url, encoding="utf-8", decode_responses=True)`. Return `self._redis`. If `redis_url` is None/empty, return `None`.

4. **`get_cached(key: str)`**: 
   - Call `_ensure_client()`. If client exists, `value = await client.get(key)`, return `json.loads(value) if value else None`.
   - If no client (memory fallback): check `_memory_ttl` for expiry with `if time.time() > self._memory_ttl.get(key, 0)`. If expired, delete from `_memory_cache` and `_memory_ttl`, return `None`. Return `self._memory_cache.get(key)`.

5. **`set_cached(key: str, value: Dict[str, Any], ttl: int = 3600)`**:
   - Call `_ensure_client()`. If client exists: `await client.setex(key, ttl, json.dumps(value))`.
   - If no client (memory fallback): `self._memory_cache[key] = value`, `self._memory_ttl[key] = time.time() + ttl`.

6. **`make_cache_key(message: str, skill_level: str, budget_brl: float)`**: EXACTLY the same as current implementation — return `f"chat:v1:{md5_hash}"` where hash is MD5 of `f"{message}:{skill_level}:{budget_brl}"`.

7. **`clear()`**: Call `_ensure_client()`. If client: `await client.flushdb()`. Then clear `self._memory_cache.clear()` and `self._memory_ttl.clear()`.

Keep the same `"""docstring"""` style and PT-BR comments as existing file.
