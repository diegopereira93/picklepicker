<task type="auto" tdd="false">
  <name>Task 3: Implement TBT optimization</name>
  <files>frontend/src/components/*</files>
  <action>
    Reduce JavaScript execution time:

    1. **Code-splitting**: Use `next/dynamic` to lazy-load heavy components:
       - Chat components (not critical for initial page load)
       - Quiz components (load on interaction)
       - Animation libraries (if any)

    2. **Defer non-essential scripts**: Move analytics, non-critical scripts to `afterInteractive`

    3. **Memoization**: Add `React.memo()` to components that re-render frequently

    Example:
    ```tsx
    // Lazy load chat (not visible on homepage)
    import dynamic from 'next/dynamic'
    const MessageBubble = dynamic(() => import('./chat/message-bubble'), {
      loading: () => <div className="animate-pulse">Loading...</div>
    })
    ```
  </action>
  <verify>
    <automated>grep for "next/dynamic" or "React.memo"</automated>
  </verify>
  <done>TBT optimization implemented</done>
</task>
