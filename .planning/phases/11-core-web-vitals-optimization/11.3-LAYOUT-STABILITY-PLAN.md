# Plan 11.3: Layout Stability & CLS Fixes

## Objective
Eliminate layout shifts to achieve CLS < 0.1. Reserve space for dynamic content, fix font loading shifts, and ensure ad/sponsored containers have fixed dimensions.

## Scope

### In Scope
- Add min-height containers for dynamic content
- Reserve space for images loading (already done in 11.1)
- Reserve space for fonts (font-display: swap handled in 11.2)
- Fix ad/sponsored content container dimensions
- Add skeleton placeholders for async content

### Out of Scope
- Redesigning layout structure
- Removing dynamic content
- New component creation (use existing patterns)

## Success Criteria
1. CLS score < 0.1 on all pages
2. No layout shifts from image loading
3. No layout shifts from font loading
4. Dynamic content areas have reserved space
5. Ad containers have explicit dimensions

## Technical Context

### Current State
- Images now have dimensions (from Plan 11.1)
- Fonts use swap display (from Plan 11.2)
- Dynamic content (product grids) may cause shifts
- Ad/sponsored content areas may not have fixed sizes

### Target State
- All dynamic content has min-height reserved
- Async components use Suspense with skeleton fallbacks
- Ad containers have explicit width/height
- No unexpected layout reflows during page lifecycle

<action>

   function Skeleton({ className, ...props }: React.HTMLAttributes<HTMLDivElement>) {

### Task 4: Verify Font Loading Layout Stability

**Objective:** Confirm fonts not causing layout shifts after 11.2 changes

**Prerequisites:** Plan 11.2 complete

**Files:**

<read_first>
- frontend/src/app/layout.tsx
</read_first>

   ```tsx
   // Should already exist from 11.2:
   const inter = Inter({
     subsets: ['latin'],
     display: 'swap',
     adjustFontFallback: true,  // Critical for CLS
     preload: true,
   });
   ```

2. Add CSS to minimize font swap impact:
   ```css
   /* In globals.css or tailwind config */
   html {
     font-feature-settings: 'liga' 1;
   }

   /* Smooth transition for font swap (optional) */
   body {
     transition: font-family 0.2s ease;
   }
   ```

3. Document font CLS prevention:
   - adjustFontFallback: true matches fallback font metrics
   - display: 'swap' prevents invisible text
   - Preload ensures font available before render
</action>
