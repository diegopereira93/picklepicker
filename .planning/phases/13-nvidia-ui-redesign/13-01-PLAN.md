# Phase 13 Plan 01: Design Tokens & Typography Foundation

**Phase:** 13 — NVIDIA UI Redesign
**Plan:** 1 of 4
**Requirements:** NV-01, NV-02, NV-03, NV-04, NV-11

## Goal

Establish the complete NVIDIA design token system in `globals.css` and load required fonts in `layout.tsx`, giving all downstream plans a stable foundation of CSS custom properties to reference.

## Context

Plans 02–04 all consume the CSS variables defined here. This plan must run first. The current `globals.css` uses shadcn/Tailwind HSL variables for a lime/amber palette. Those variables are replaced with the NVIDIA hex palette while keeping the Tailwind directives so existing Tailwind layout utilities continue to work. The current `layout.tsx` loads Inter from Google Fonts; NVIDIA-EMEA is loaded via `@font-face` with Arial fallback, and Font Awesome 6 is added via CDN `<link>` tag only — no npm install.

### Task 2: Add Font Awesome CDN link and NVIDIA font-family to layout.tsx

**File:** `frontend/src/app/layout.tsx`
**Action:** edit

Two changes only — do not alter any other logic:

1. Remove the `Inter` next/font import and the `inter` constant. Remove `inter.variable` from the `<html>` className. Remove `font-sans` from the body className.

2. Add a Font Awesome 6 CDN `<link>` tag inside `<head>`. Since Next.js App Router does not use an explicit `<head>` tag in RootLayout by default, add it via the `metadata` object's `other` field or by adding an explicit `<head>` element. The safest approach: add `<head><link rel="stylesheet" href="https://kit.fontawesome.com/a077f84a44.css" crossOrigin="anonymous" /></head>` before `<body>` inside the `<html>` element. Use the public Font Awesome 6 Free CDN instead if no kit ID is available: `href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.0/css/all.min.css"`.

3. Add `style={{ fontFamily: "'NVIDIA-EMEA', Arial, Helvetica, sans-serif" }}` to the `<html>` element alongside the existing `lang` and `suppressHydrationWarning` attributes.

The body className changes to: `"min-h-screen bg-background text-foreground antialiased flex flex-col"` (remove `font-sans`, keep the rest for Tailwind compat).

**Verification:** `grep -i "fontawesome\|font-awesome" frontend/src/app/layout.tsx` returns a match. `grep "NVIDIA-EMEA" frontend/src/app/layout.tsx` returns a match. `grep "inter\|Inter" frontend/src/app/layout.tsx` returns no match.
