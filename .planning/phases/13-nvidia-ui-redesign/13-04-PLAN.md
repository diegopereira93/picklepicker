# Phase 13 Plan 04: Pages & Product Cards

**Phase:** 13 — NVIDIA UI Redesign
**Plan:** 4 of 4
**Requirements:** NV-06, NV-09, NV-10, NV-12

## Goal

Apply NVIDIA styling to the home page, product catalog, product card component, and skeleton — establishing dark/light section alternation, the NVIDIA card contract with green title underline, and the responsive 3/2/1 column catalog grid.

## Context

Depends on Plans 01, 02, 03 (tokens, buttons, layout shell all in place). This plan uses the `.nv-dark-section`, `.nv-light-section`, `.nv-container`, `.nv-section`, and `.nv-section-hero` classes from Plan 01/03. It adds new `.nv-product-card`, `.nv-catalog-grid`, and `.nv-hero` CSS classes to `globals.css`. The `card.tsx` component keeps its existing prop API (all sub-components exported unchanged) but its default className values are rewritten to NVIDIA spec. Checkpoint at end for visual QA.
