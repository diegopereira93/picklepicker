# Phase 13 Plan 03: Navigation & Layout Shell

**Phase:** 13 — NVIDIA UI Redesign
**Plan:** 3 of 4
**Requirements:** NV-07, NV-09, NV-10

## Goal

Restyle the site header to a sticky black NVIDIA navigation bar with uppercase links and hamburger collapse at ≤1024px, restyle the footer for responsive stacking, and add the responsive breakpoint CSS and section-padding scale to `globals.css`.

## Context

Depends on Plan 01 (CSS tokens). Can run in parallel with Plan 02 since they touch different files. The current header uses shadcn `Sheet` for the mobile drawer — keep that component but restyle it with NVIDIA colors. The current header collapses at `md` (768px) — change to ≤1024px per NV-07 spec. All nav link text becomes uppercase 14px 700. ThemeToggle is removed (NVIDIA design is always dark-dominant — no light/dark toggle). The footer component path needs to be read before editing — check it exists at `frontend/src/components/layout/footer.tsx`.

## Tasks

### Task 1: Restyle header.tsx to NVIDIA navigation bar

**File:** `frontend/src/components/layout/header.tsx`
**Action:** edit

**Changes to make:**

1. Remove the `ThemeToggle` import and its `<ThemeToggle />` usage — NVIDIA design has no theme toggle.

2. Update `navLinks` labels to uppercase display (the CSS will handle `text-transform`, but also update the label strings to match NVIDIA nav copy convention):
   ```js
   const navLinks = [
     { href: "/", label: "HOME" },
     { href: "/paddles", label: "CATÁLOGO" },
   ]
   ```

3. Replace the `<header>` element className:
   - FROM: `"sticky top-0 z-50 w-full border-b bg-background/95 backdrop-blur supports-[backdrop-filter]:bg-background/60"`
   - TO: `"nv-nav sticky top-0 z-50 w-full"`

4. Replace the inner container `<div>` className:
   - FROM: `"container flex h-14 items-center"`
   - TO: `"nv-container flex h-14 items-center"`

5. Replace the logo `<Link>` className and content:
   - FROM: `"mr-6 flex items-center space-x-2"` with inner `<span className="font-bold text-xl text-primary">PickleIQ</span>`
   - TO: `"nv-nav-logo mr-6 flex items-center"` with inner `<span className="nv-nav-brand">PickleIQ</span>`

6. Replace desktop nav `<nav>` className:
   - FROM: `"hidden md:flex items-center space-x-6 text-sm font-medium flex-1"`
   - TO: `"nv-nav-links hidden items-center space-x-6 flex-1"` — Note: hide/show breakpoint is now controlled by CSS media query `.nv-nav-links` (not Tailwind `md:flex`), so remove `md:` and add `nv-nav-links` which the CSS will show at >1024px.

7. Replace each nav link className:
   - FROM: `"text-foreground/60 transition-colors hover:text-foreground"`
   - TO: `"nv-nav-link"`

8. Replace desktop CTA `<div>` className:
   - FROM: `"hidden md:flex items-center space-x-2 ml-auto"`
   - TO: `"nv-nav-cta hidden items-center space-x-2 ml-auto"` (CSS shows at >1024px)

9. Keep the `Button` component for the CTA link — it will pick up NVIDIA styles from Plan 02's button rewrite. Remove `size="sm"` from the CTA Button (use default size for proper 11px 13px padding).

10. Replace the mobile hamburger `<div>` className:
    - FROM: `"flex md:hidden ml-auto"`
    - TO: `"nv-nav-mobile flex ml-auto"` (CSS shows only at ≤1024px)

11. The `SheetContent` className: add `"nv-nav-overlay"` to give it black background.

12. Each mobile nav link className in `SheetContent`:
    - FROM: `"text-foreground/80 text-lg font-medium transition-colors hover:text-foreground"`
    - TO: `"nv-nav-link-mobile"`

**Verification:** `grep -c "nv-nav" frontend/src/components/layout/header.tsx` returns at least 5.

### Task 2: Restyle footer.tsx for responsive stacking

**File:** `frontend/src/components/layout/footer.tsx`
**Action:** edit

First check the file content. The footer likely uses Tailwind grid/flex classes. Replace the outermost `<footer>` element className with `"nv-footer nv-dark-section"`. Replace any inner container div with `"nv-container"`. Replace any multi-column grid class (`grid-cols-*`) with `"nv-footer-grid"`. Replace copyright/legal text classes with `"nv-caption-small"` (which renders at 12px #757575 per spec).

**Verification:** `grep "nv-footer" frontend/src/components/layout/footer.tsx` returns at least 1 match.
